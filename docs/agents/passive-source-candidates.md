## Current outcome

Kimi Code (`kimi_code`) and Pi / Oh My Pi (`pi`) have passed the passive-reader gate for this implementation slice. They use sanitized fixtures, sync-twice and rewrite/append coverage, source-scoped cursors, explicit token-quality contracts, and privacy-bounded normalization. Reasonix and the remaining candidates stay monitor-only; `llmusage source-status` may report their roots and blocked parser state, but sync must not write usage rows for them.
